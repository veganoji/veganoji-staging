「お肉なしで、力は出ない」？
3つの方向性

力チャプターの作り直し案を3つ。 どれも内容（草食動物の強さ → 植物のタンパク質 → 「栄養はタンパク質だけじゃない」比較表 → アスリート5名）は同じで、見せ方とトーンが違います。 1つ選んでもらえれば、本ページ（/vegan-to-karada/）に組み込みます。

VERSION 1

強さ図鑑 — STRENGTH DEX

ゲームの「図鑑／キャラ選択」みたいに、ノリよく派手に。 動物はステータス・カード（タップで「食べてるのは植物」を公開）、タンパク質はランキング・バー、比較表はスポーツの「スコアボード」風。 数字とインパクト重視。

プレビューを見る → VERSION 2

栄養レントゲン — NUTRITION X-RAY

落ち着いた、エディトリアル（雑誌）寄り。 タンパク質神話を素早く片づけて、本題へ：「足りてないのはタンパク質じゃなく、食物繊維（不足94%）」。 比較表は静かな"レントゲン"表で、行をタップすると隠れた事実が開く。 データで説得。

プレビューを見る → VERSION 3

王子と歩く — OJI WALK-THROUGH

いまの /vegan-to-karada/ といちばん地続きの、あたたかい王子ナビ型。 動物は「へぇ〜カード」、比較表は信号機（🟢🟡🔴）パネルで「王子のひとこと」が隠れた事実をやさしく開く。 アスリートはポラロイド風カード。 親しみ重視。
